# SibbPhrase (シブフレーズ)

> [!NOTE]
> このプロジェクトのコード、README、コメントを含むドキュメントは、AI に作成してもらったものです。
> 私自身は Android 開発を学習中であり、内容を確認・検証しながら公開していますが、実装や説明に誤りや改善の余地がある可能性があります。

パスワードなどの機密情報を安全に暗号化・復号し、家族間で共有できるローカルファーストの Android アプリです。

外部サーバーへの通信を一切行わず、すべての処理がデバイス上で完結します。
暗号化したテキストをメールやメッセージアプリに貼り付けて送ることで、安全に情報を渡すことができます。

あらかじめ家族間で決めた「合言葉」をそれぞれのスマホに設定しておくことで、同じ合言葉を持つ端末間でのみ、暗号化された文字を元の正しいパスワードに復元できます。

## 特徴

- 外部サーバー通信なしで、すべて端末内で完結
- AES-256-GCM による暗号化・復号
- 家族で共有する「合言葉」方式
- QRコードでの送受信に対応
- 生体認証によるアプリロックに対応
- 自動削除・表示時間などを設定可能
- 初回起動時の案内画面付き

## できること

### 暗号化
入力したテキストを暗号化して、コピーや共有ができます。

### 復号
受け取った暗号文を、設定済みの合言葉を使って復元できます。

### 合言葉の管理
家族で共通の合言葉を設定し、QRコードで受け渡しできます。

### 設定
テーマ、自動削除秒数、復元表示秒数、生体認証の有無などを調整できます。

## 使い方

1. 初回起動時に案内に従って初期設定を行います。
2. 家族それぞれの端末に、同じ合言葉を設定します。
3. 共有したい情報を入力して暗号化します。
4. 生成された暗号文をメールやメッセージで相手に送ります。
5. 相手側で復号すると、元の内容を確認できます。

## 注意事項

- 合言葉は、端末ごとに正しく設定してください。
- 暗号化した文字列は、復号に必要な情報を含みます。外部に送る際は、送信先を必ず確認してください。
- 本アプリはローカル完結型ですが、送信方法そのものはユーザーご自身で選ぶ必要があります。
- 合言葉を忘れると復号できなくなります。

## プライバシー

SibbPhrase は、外部サーバーへの通信を行いません。
入力内容や合言葉は、端末上で管理されます。

## 開発環境 (Tech Stack)

* **Language**: Kotlin
* **UI**: Jetpack Compose (単一アクティビティ構造)
* **Architecture**: MVVM / Unidirectional Data Flow (UDF)
* **DI (Dependency Injection)**: Hilt
* **Jetpack Libraries**: 
  * Navigation 3
  * DataStore Preferences (設定・生体認証状態の永続化)
  * CameraX (QRコードスキャン用)
* **Security & Crypto**:
  * Jetpack Biometric (生体認証)
  * Android Keystore / EncryptedSharedPreferences
* **Other Libraries**:
  * ML Kit Barcode Scanning (高速なQRコード解析)
  * ZXing (QRコード生成)
